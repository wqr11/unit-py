## Start
### Install
```
git clone https://github.com/wqr11/unit-py && cd ./unit-py
```
### Create venv
```
python3 -m venv venv
```
## Development
### Start db
```
docker compose up
```
#### Bash
```bash
source ./venv/bin/activate && python3 ./main.py
```
#### Fish
```fish
source ./venv/bin/activate.fish && python3 ./main.py
```
